# EMS IBR

### How to install

1. Clone the reop
2. Create a virtual environment
3. Add .env with all fields
4. Setup Database
5. Run `pip install -r requirements.txt`
6. Run `python manage.py migrate`
7. Run `python manage.py runserver`
8. Open Browser to `localhost:8000`

### .ENV EXAMPLE

```
DEBUG=
SECRET_KEY=
DB_NAME=
DB_USER=
DB_PASSWORD=
```
